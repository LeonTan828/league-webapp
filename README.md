# How project started

## for client
1. `npm create vite@latest`
project name: client
react, typescript
update and change app.tsx and use function code (see early git commit)

2. `npm install sass`


## for server
1. create new folder
2. `npm init`
mostly default, except change index to typescript, module for type, and MIT license


## To run

1. cd to client
`npm run dev`
